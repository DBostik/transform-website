// turbo-all

## Starting Development

1. Start the dev server:
```bash
npm run dev
```

## Building for Production

1. Build the production bundle:
```bash
npm run build
```

2. Preview the production build:
```bash
npm run preview
```

## Git Operations

1. Check status:
```bash
git status
```

2. Stage all changes:
```bash
git add .
```

3. View recent commits:
```bash
git log -n 5 --oneline
```
